Project on Console

To use Visual Studio, first you need to make a new folder on your desktop and name it "Visual Studio files". Then, you need to create another folder inside the folder you just made and then you need to open Visual Studio and click on files and then click on open folder, then open the new folder you just made, next you need to set up the html and then work on your project!

The console screen is very important, this is because if your project fors not work, you can look at the console to see what the problem is, and then fix it. To open the console screen, you need to right click on the website and then click on inspect and then click on Console. To print things on the console, you need to us console.log(""). then you can put whatever youj want. If it is a variable, then you do not need the quotations.
